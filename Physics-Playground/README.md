## Physics Playground
This repo consists of my ongoing physics playground project built using C++ and SFML.
This project is currenty in development phase.

---

## Features
-Object spawing on mouse click  
-Projectile motion simulation  
-Gravity and rebound physics  
-Loss of object velocity with impulse and impact
